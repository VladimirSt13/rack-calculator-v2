# Опис результатів на сторінці "Розрахунок стелажа" (RackPage)

## 📊 Структура результатів

Результати відображаються **тільки після** натискання кнопки "Розрахувати" і успішного відповіді від сервера.

### Стани результатів

| Стан        | Опис                                   | Компонент                    |
| ----------- | -------------------------------------- | ---------------------------- |
| **Idle**    | Форма заповнена, розрахунок не початий | `EmptyState`                 |
| **Loading** | Запит на сервер, дані завантажуються   | `ResultsSkeleton`            |
| **Ready**   | Дані отримані, показані результати     | `RackResults`                |
| **Error**   | Помилка сервера або валідації          | `EmptyState` з повідомленням |

## 1️⃣ PreambleCard — Картка коротких результатів

**Розташування:** Верхня картка, одразу після заголовка сторінки

**Структура:**

```
┌────────────────────────────────────────────┐
│ ✓ Результати розрахунку                   │
├────────────────────────────────────────────┤
│                                            │
│  Прольотів    Балок      Опор      Вартість│
│     5         20 шт      12 шт    1234 ₴   │
│                                            │
└────────────────────────────────────────────┘
```

### Елементи

| Елемент       | Тип       | Дані                                  | Формат                    |
| ------------- | --------- | ------------------------------------- | ------------------------- |
| **Іконка**    | Icon      | `CheckCircle2`                        | w-5 h-5, text-success     |
| **Заголовок** | CardTitle | "Результати розрахунку"               | text-base                 |
| **Прольотів** | Metric    | `result.spans.reduce()`               | число                     |
| **Балок**     | Metric    | `result.components.beams.reduce()`    | число + "шт"              |
| **Опор**      | Metric    | `result.components.supports.reduce()` | число + "шт"              |
| **Вартість**  | Price     | `result.pricing.zero`                 | `PriceDisplay`, size="lg" |

- Desktop (lg): 4 колонки
- Mobile: 2 колонки
- Відступи: gap-4

### Стан комплекту

```typescript
interface RackSetItem {
  rackId: string;
  name: string;
  quantity: number;
  result: RackCalculationResult;
}

interface RackSetState {
  items: RackSetItem[];
  totalZero: number;
  totalWithoutIsolators: number;
}
```

### Дії

| Дія                     | Опис                                   | Компонент                   |
| ----------------------- | -------------------------------------- | --------------------------- |
| **Додати стелаж**       | Додає стелаж до комплекту з quantity=1 | `addRackToSet(result)`      |
| **Збільшити кількість** | +1 до quantity                         | `incrementQuantity(rackId)` |
| **Зменшити кількість**  | -1 до quantity (min=1)                 | `decrementQuantity(rackId)` |
| **Видалити стелаж**     | Видаляє стелаж з комплекту             | `removeRackFromSet(rackId)` |
| **Очистити комплект**   | Видаляє всі стелажі                    | `clearSet()`                |
| **Зберегти комплект**   | Відкриває модальне вікно               | `openSaveModal()`           |

## 📦 Тип даних результату

```typescript
interface RackCalculationResult {
  name: string; // Повна назва стелажа
  description?: string; // Опис (опціонально)
  tableHtml?: string; // HTML-таблиця (для експорту)
  components: RackComponents; // Компоненти (див. нижче)
  pricing: RackPricing; // Ціни (base, withoutIsolators, zero)
  form?: RackFormState; // Дані форми для редагування
  spans: SpanItem[]; // Прольоти для підрахунку кількості
  rackSetId?: string; // ID комплекту в БД
  revisionId?: string; // ID ревізії
}
```

### RackComponents структура

```typescript
type RackComponents = {
  supports: SupportItem[];
  beams: BeamItem[];
  verticalStands?: VerticalStandItem;
  braces?: BraceItem;
  isolators?: IsolatorItem;
};

interface SupportItem {
  type: 'edge' | 'intermediate';
  quantity: number;
  price: number; // Ціна за одиницю
  total: number; // Загальна вартість (quantity × price)
}

interface BeamItem {
  type: string; // напр. '600', '750'
  length: number; // довжина в мм
  quantity: number;
  price: number;
  total: number;
}

interface VerticalStandItem {
  type: string; // напр. '1190', '632'
  quantity: number;
  price: number;
  total: number;
}

interface BraceItem {
  quantity: number;
  price: number;
  total: number;
}

interface IsolatorItem {
  quantity: number;
  price: number;
  total: number;
}
```

### RackPricing структура

```typescript
interface RackPricing {
  base: number; // Базова ціна (сума всіх компонентів)
  withoutIsolators: number; // Ціна без ізоляторів
  zero: number; // Нульова ціна (base × 1.44)
}
```

**Приклад:**

```javascript
{
  supports: [
    { type: 'edge', quantity: 4, price: 100, total: 400 },
    { type: 'intermediate', quantity: 6, price: 80, total: 480 }
  ],
  beams: [
    { type: '600', length: 600, quantity: 10, price: 200, total: 2000 }
  ],
  verticalStands: { type: '1190', quantity: 8, price: 150, total: 1200 },
  braces: { quantity: 4, price: 50, total: 200 },
  isolators: { quantity: 8, price: 25, total: 200 }
}

pricing: {
  base: 4480,           // 400 + 480 + 2000 + 1200 + 200 + 200
  withoutIsolators: 4280, // 4480 - 200
  zero: 6451            // 4480 × 1.44
}
```

**Компонент:** `RackSetCard`

### Вміст

1. **Заголовок:** "Комплект стелажів" з іконкою

  ├─ Форма (RackForm)
  │    ├─ levels, rows, beamsPerRow
  │    ├─ supportType, verticalStandType
  │    └─ spans[] (type, quantity)
  │
  ├─ Кнопка "Розрахувати"
  │    └─ POST /api/rack/calculate
  │
  ├─ Результати (RackResults)
  │    ├─ PreambleCard (коротко: прольоти, балки, опори, вартість)
  │    ├─ RackNameCard (повна назва)
  │    └─ ComponentsTableCard
  │         ├─ Таблиця компонентів (з цінами та сумами)
  │         ├─ Блок цін (базова, нульова, без ізоляторів)
  │         └─ Кнопка "Додати до комплекту"
  │              └─ Додає в SetPanel
  │
  ├─ SetPanel (права панель)
  │    ├─ Список стелажів з контролем кількості
  │    ├─ Підсумкова вартість
  │    └─ Кнопка "Зберегти комплект"
  │         └─ Відкриває SaveSetModal
  │
  └─ SaveSetModal (модальне вікно)
       ├─ Склад комплекту (з компонентами)
       ├─ Поля: Назва об'єкту, Примітка
       ├─ Підсумкова вартість
       └─ Кнопки: Зберегти / Скасувати
