# Описание формы "Расчёт стеллажа" (RackPage)

## 📐 Расположение элементов формы

### Общая структура

```
┌─────────────────────────────────────────────────────┐
│  Параметры стелажа                      [✕ Очистить]│
├─────────────────────────────────────────────────────┤
│  ┌─ Геометрия ──────────────────────────────────┐  │
│  │                                               │  │
│  │  Количество этажей      [1]  (input, 1-10)   │  │
│  │                                               │  │
│  │  Вертикальная опора     [▼ Выберіть...]      │  │
│  │                         (disabled если 1 этаж)│  │
│  │                                               │  │
│  │  Опора                  [▼ Выберіть...]      │  │
│  │                                               │  │
│  │  Количество рядов       [1]  (input, 1-4)    │  │
│  │                                               │  │
│  │  Балок в ряду           [2]  (input, 2-4)    │  │
│  └───────────────────────────────────────────────┘  │
│                                                     │
│  ┌─ Пролёты ────────────────────────────────────┐  │
│  │  [+] Додати проліт                           │  │
│  │                                               │  │
│  │  ┌─────────────────────────────────────────┐ │  │
│  │  │ [▼ Проліт...] [1] [−]                   │ │  │
│  │  └─────────────────────────────────────────┘ │  │
│  │                                               │  │
│  │  ┌─────────────────────────────────────────┐ │  │
│  │  │ [▼ Проліт...] [5] [−]                   │ │  │
│  │  └─────────────────────────────────────────┘ │  │
│  │                                               │  │
│  │  ... (список пролётов)                        │  │
│  └───────────────────────────────────────────────┘  │
└─────────────────────────────────────────────────────┘
```

### Детальное описание расположения

#### Заголовок формы
- **Слева**: "Параметри стелажа" (text-lg font-semibold)
- **Справа**: Кнопка "Очистити" с иконкой ✕ (text-xs, muted-foreground)

#### Секция "Геометрия" (FormSection)

Элементы расположены **вертикально**, один под другим:

1. **Количество этажей**
   - Label слева, input справа
   - Input: числовое поле, ширина ~9 символов (charWidth={9})
   - Диапазон: 1–10

2. **Вертикальная опора**
   - Label слева, select справа
   - Select: полный ширины контейнера
   - Плейсхолдер: "Виберіть..."
   - **Disabled**: когда `floors === 1` (автоматически сбрасывается в "")

3. **Опора**
   - Label слева, select справа
   - Select: полный ширины контейнера
   - Плейсхолдер: "Виберіть..."

4. **Количество рядов**
   - Label слева, input справа
   - Input: числовое поле, ширина ~9 символов
   - Диапазон: 1–4

5. **Балок в ряду**
   - Label слева, input справа
   - Input: числовое поле, ширина ~9 символов
   - Диапазон: 2–4

#### Секция "Пролёты" (FormSection)

1. **Кнопка добавления**
   - Расположение: горизонтально (flex row)
   - Элементы: `[+]` (IconButton) + "Додати проліт" (text-sm muted-foreground)
   - Отступ снизу: mb-2

2. **Список пролётов** (SpanList)
   - Каждый пролёт — карточка с border и rounded-md
   - **Grid-раскладка**: `grid-cols-[1fr_auto_auto]` (select | количество | удалить)
   - Отступы: p-2.5, gap-2

   **Структура одного пролёта:**
   ```
   ┌────────────────────────────────────────────┐
   │ [▼ Выберіть проліт...]  [1]  [−]          │
   │  ↑ flex-1, min-w-[120px] ↑ w-[72px]       │
   └────────────────────────────────────────────┘
   ```

   - **Select пролёта**: 
     - Ширина: flex-1, min-w-[120px]
     - Высота: h-8
     - Опции отображаются как "{label} мм"
   
   - **Input количества**:
     - Ширина: 72px (w-[72px])
     - Выравнивание: по центру (text-center)
     - Шрифт: моноширинный (font-mono)
     - Диапазон: min=1
   
   - **Кнопка удаления**:
     - IconButton с иконкой "−" (Minus)
     - Вариант: "icon"

3. **Пустое состояние**
   - Если пролётов нет: текст "Немає доданих прольотів"
   - Расположение: по центру, p-4, text-sm, muted-foreground

## 📋 Элементы формы и их ограничения

### Поля ввода (Inputs)

| Поле | Тип | Min | Max | Default | Ширина | Обязательность |
|------|-----|-----|-----|---------|--------|----------------|
| Количество этажей | number | 1 | 10 | 1 | 9ch | ✅ |
| Количество рядов | number | 1 | 4 | 1 | 9ch | ✅ |
| Балок в ряду | number | 2 | 4 | 2 | 9ch | ✅ |
| Количество (пролёт) | number | 1 | — | 1 | 72px | ✅ |

### Выпадающие списки (Selects)

| Поле | Источник данных | Плейсхолдер | Ограничения | Обязательность |
|------|-----------------|-------------|-------------|----------------|
| Вертикальная опора | `verticalSupportsComponents` (API) | "Виберіть..." | Disabled если этажей = 1 | ✅ Если этажей > 1 |
| Опора | `supportsComponents` (API) | "Виберіть..." | — | ✅ |
| Пролёт (в списке) | `spansComponents` (API) | "Виберіть проліт..." | — | ✅ (фактически) |

### Кнопки

| Кнопка | Иконка | Расположение | Действие |
|--------|--------|--------------|----------|
| Додати проліт | Plus (+) | Секция "Пролёты", сверху | Добавляет новый пролёт в список |
| Видалити проліт | Minus (−) | Справа у каждого пролёта | Удаляет конкретный пролёт |
| Очистити | X (✕) | Справа у заголовка формы | Сбрасывает форму в начальное состояние |

## 🔧 Валидация (Zod схема)

```typescript
const rackFormSchema = z.object({
  floors: z.coerce.number().min(1).max(10),
  verticalSupports: z.string().optional(),
  supports: z.string().min(1, "Оберіть тип опори"),
  rows: z.coerce.number().min(1).max(4),
  beamsPerRow: z.coerce.number().min(2).max(4),
});
```

### Правила валидации

| Поле | Правило | Сообщение об ошибке |
|------|---------|---------------------|
| floors | 1 ≤ value ≤ 10 | стандартное Zod |
| verticalSupports | optional | — |
| supports | min 1 символ | "Оберіть тип опори" |
| rows | 1 ≤ value ≤ 4 | стандартное Zod |
| beamsPerRow | 2 ≤ value ≤ 4 | стандартное Zod |

## 📦 Состояния формы

### Начальное состояние (default)
```typescript
{
  floors: 1,
  verticalSupports: "",
  supports: "",
  rows: 1,
  beamsPerRow: 2,
  spans: []  // пустой список пролётов
}
```

### Состояние "Нет пролётов"
- Отображается текст: "Немає доданих прольотів"
- Кнопка [+] доступна для добавления

### Состояние "Загрузка компонентов"
- Показывается спиннер + текст "Завантаження комплектуючих..."
- Форма скрыта до завершения загрузки

### Состояние "1 этаж"
- Поле "Вертикальная опора" disabled
- При изменении на 1 этаж: значение сбрасывается в ""
