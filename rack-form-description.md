# Опис форми "Розрахунок стелажа" (RackPage)

## 📐 Розташування елементів форми

### Загальна структура

```
┌─────────────────────────────────────────────────────┐
│  Параметри стелажа                      [✕ Очистити]│
├─────────────────────────────────────────────────────┤
│  ┌─ Геометрія ──────────────────────────────────┐  │
│  │                                               │  │
│  │  Кількість поверхів       [1]  (input, 1-10) │  │
│  │                                               │  │
│  │  Вертикальна опора        [▼ Виберіть...]    │  │
│  │                         (disabled якщо 1 поверх)│  │
│  │                                               │  │
│  │  Опора                  [▼ Виберіть...]      │  │
│  │                                               │  │
│  │  Кількість рядів          [1]  (input, 1-4)  │  │
│  │                                               │  │
│  │  Балок в ряду             [2]  (input, 2-4)  │  │
│  └───────────────────────────────────────────────┘  │
│                                                     │
│  ┌─ Прольоти ───────────────────────────────────┐  │
│  │  [+] Додати проліт                           │  │
│  │                                               │  │
│  │  ┌─────────────────────────────────────────┐ │  │
│  │  │ [▼ Проліт...] [1] [−]                   │ │  │
│  │  └─────────────────────────────────────────┘ │  │
│  │                                               │  │
│  │  ┌─────────────────────────────────────────┐ │  │
│  │  │ [▼ Проліт...] [5] [−]                   │ │  │
│  │  └─────────────────────────────────────────┘ │  │
│  │                                               │  │
│  │  ... (список прольотів)                        │  │
│  └───────────────────────────────────────────────┘  │
└─────────────────────────────────────────────────────┘
```

### Детальний опис розташування

#### Заголовок форми

- **Зліва**: "Параметри стелажа" (text-lg font-semibold)
- **Справа**: Кнопка "Очистити" з іконкою ✕ (text-xs, muted-foreground)

#### Секція "Геометрія" (FormSection)

Елементи розташовані **вертикально**, один під одним:

1. **Кількість поверхів**
   - Label зліва, input справа
   - Input: числове поле, ширина ~9 символів (charWidth={9})
   - Діапазон: 1–10

2. **Вертикальна опора**
   - Label зліва, select справа
   - Select: повний ширини контейнера
   - Плейсхолдер: "Виберіть..."
   - **Disabled**: коли `floors === 1` (автоматично скидається в "")

3. **Опора**
   - Label зліва, select справа
   - Select: повний ширини контейнера
   - Плейсхолдер: "Виберіть..."

4. **Кількість рядів**
   - Label зліва, input справа
   - Input: числове поле, ширина ~9 символів
   - Діапазон: 1–4

5. **Балок в ряду**
   - Label зліва, input справа
   - Input: числове поле, ширина ~9 символів
   - Діапазон: 2–4

#### Секція "Прольоти" (FormSection)

1. **Кнопка додавання**
   - Розташування: горизонтально (flex row)
   - Елементи: `[+]` (IconButton) + "Додати проліт" (text-sm muted-foreground)
   - Відступ знизу: mb-2

2. **Список прольотів** (SpanList)
   - Кожен проліт — картка з border та rounded-md
   - **Grid-розкладка**: `grid-cols-[1fr_auto_auto]` (select | кількість | видалити)
   - Відступи: p-2.5, gap-2

   **Структура одного прольоту:**

   ```
   ┌────────────────────────────────────────────┐
   │ [▼ Виберіть проліт...]  [1]  [−]          │
   │  ↑ flex-1, min-w-[120px] ↑ w-[72px]       │
   └────────────────────────────────────────────┘
   ```

   - **Select прольоту**:
     - Ширина: flex-1, min-w-[120px]
     - Висота: h-8
     - Опції відображаються як "{label} мм"

   - **Input кількості**:
     - Ширина: 72px (w-[72px])
     - Вирівнювання: по центру (text-center)
     - Шрифт: моноширинний (font-mono)
     - Діапазон: min=1

   - **Кнопка видалення**:
     - IconButton з іконкою "−" (Minus)
     - Варіант: "icon"

3. **Пустий стан**
   - Якщо прольотів немає: текст "Немає доданих прольотів"
   - Розташування: по центру, p-4, text-sm, muted-foreground

## 📋 Елементи форми та їх обмеження

### Поля введення (Inputs)

| Поле               | Тип    | Min | Max | Default | Ширина | Обов'язковість |
| ------------------ | ------ | --- | --- | ------- | ------ | -------------- |
| Кількість поверхів | number | 1   | 10  | 1       | 9ch    | ✅             |
| Кількість рядів    | number | 1   | 4   | 1       | 9ch    | ✅             |
| Балок в ряду       | number | 2   | 4   | 2       | 9ch    | ✅             |
| Кількість (проліт) | number | 1   | —   | 1       | 72px   | ✅             |

### Випадаючі списки (Selects)

| Поле              | Джерело даних                      | Плейсхолдер          | Обмеження                  | Обов'язковість       |
| ----------------- | ---------------------------------- | -------------------- | -------------------------- | -------------------- |
| Вертикальна опора | `verticalSupportsComponents` (API) | "Виберіть..."        | Disabled якщо поверхів = 1 | ✅ Якщо поверхів > 1 |
| Опора             | `supportsComponents` (API)         | "Виберіть..."        | —                          | ✅                   |
| Проліт (в списку) | `spansComponents` (API)            | "Виберіть проліт..." | —                          | ✅ (фактично)        |

### Кнопки

| Кнопка          | Іконка    | Розташування              | Дія                               |
| --------------- | --------- | ------------------------- | --------------------------------- |
| Додати проліт   | Plus (+)  | Секція "Прольоти", зверху | Додає новий проліт до списку      |
| Видалити проліт | Minus (−) | Справа у кожного прольоту | Видаляє конкретний проліт         |
| Очистити        | X (✕)     | Справа у заголовка форми  | Скидає форму до початкового стану |

## 🔧 Валідація (Zod схема)

```typescript
const rackFormSchema = z.object({
  floors: z.coerce.number().min(1).max(10),
  verticalSupports: z.string().optional(),
  supports: z.string().min(1, 'Оберіть тип опори'),
  rows: z.coerce.number().min(1).max(4),
  beamsPerRow: z.coerce.number().min(2).max(4),
});
```

### Правила валідації

| Поле             | Правило        | Повідомлення про помилку |
| ---------------- | -------------- | ------------------------ |
| floors           | 1 ≤ value ≤ 10 | стандартне Zod           |
| verticalSupports | optional       | —                        |
| supports         | min 1 символ   | "Оберіть тип опори"      |
| rows             | 1 ≤ value ≤ 4  | стандартне Zod           |
| beamsPerRow      | 2 ≤ value ≤ 4  | стандартне Zod           |

## 📦 Стани форми

### Початковий стан (default)

```typescript
{
  floors: 1,
  verticalSupports: "",
  supports: "",
  rows: 1,
  beamsPerRow: 2,
  spans: []  // пустий список прольотів
}
```

### Стан "Немає прольотів"

- Відображається текст: "Немає доданих прольотів"
- Кнопка [+] доступна для додавання

### Стан "Завантаження компонентів"

- Показується спіннер + текст "Завантаження комплектуючих..."
- Форма прихована до завершення завантаження

### Стан "1 поверх"

- Поле "Вертикальна опора" disabled
- При зміні на 1 поверх: значення скидається в ""
